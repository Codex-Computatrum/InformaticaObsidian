# Operatore `instanceof`
>[!important]
L'operatore `instanceof` in Java viene utilizzato per verificare se un oggetto è un'istanza di una determinata classe.
L'espressione `rif instanceof Class`
Restituisce true se:
> - `rif` è  sottotipo di `Class`. 
> - `rif` non è `null`.
- 
^instanceof
